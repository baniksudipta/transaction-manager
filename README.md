# Transaction Manager

## How to run locally
Ensure you have Go 1.22 installed.
```bash
go run src/main.go
```

## How to run using Docker
You can easily start the application via Docker using the provided `run.sh` script:
```bash
chmod +x run.sh
./run.sh
```

## How to execute tests
Run the following command from the root of the project to execute all unit tests:
```bash
go test ./...
```
